课程大纲

W1
破冰 — 理解 AI 编程的工作方式
聚焦：需求描述 · 提示词迭代
第 1 节
Vibe Coding 是什么，工具怎么装 →
目标：说清楚 Vibe Coding 与传统编程的区别，完成所有工具安装
  • Vibe Coding 理念你是产品经理，AI 是程序员——你不写代码，你管理 AI 写代码
  • 工具准备AI 对话工具三选一：Qoder CN（国内可用）/ Claude / Cursor
  • Agent Skill安装所选工具推荐的 Agent Skill，让 AI 更懂你的项目
  • 认识小程序安装微信开发者工具，了解小程序是什么、能做什么
  • 动手实操新建项目，运行官方示例，体验从 0 到跑起来的成就感
第 2 节
用 AI 生成图文列表页 + 页面跳转
目标：向 AI 描述页面需求，实现首页图文列表与点击跳转详情页
  • 页面结构一个页面由哪些文件组成，各自负责什么（结构 / 样式 / 逻辑）
  • 提示词技巧从"帮我做一个好看的页面"升级到精确描述卡片布局
  • 模块化思维把需求拆成"列表展示"和"详情查看"两个独立任务
  • 数据驱动与跳转描述多条数据循环展示 + 点击跳转详情页
  • 动手实操用 AI 生成首页图文列表，实现点击卡片跳转详情页
W2
迭代与组合 — 让小程序成型
聚焦：截图反馈 · 样式迭代 · API 概念 · 调试入门
第 3 节
详情页美化 + 导航框架搭建
目标：用截图迭代出满意的视觉效果，搭建完整的 App 导航结构
  • 迭代工作流"先生成 → 看到问题 → 针对性修改提示词"的循环方法
  • 截图沟通法截图发给 AI 并标注修改点，效率远高于纯文字描述
  • 样式与素材用自然语言调整颜色、间距、圆角，替换为真实素材
  • 从单页到 App向 AI 描述底部导航栏，整合为完整 App
  • 动手实操完善详情页视觉效果，添加导航栏，整合为完整 App
第 4 节
微信登录 + 数据存储
目标：实现微信登录，根据登录状态展示不同界面，数据存入云端不丢失
  • API 概念什么是接口，AI 怎么知道调用哪个
  • 登录流程用户点击 → 弹出授权 → 获取信息 → 显示头像
  • 条件逻辑用"如果…就…"描述：未登录显示按钮，已登录显示信息
  • 数据库入门用"表格"理解数据库，描述"登录时保存、打开时读取"
  • 调试技巧代码报错？把错误信息直接发给 AI 让它修复
  • 动手实操实现登录 + 个人中心，数据存入云数据库
W3
交付 — 从开发到上线
聚焦：表单设计 · 全流程联调 · 上线发布
第 5 节
联系方式表单 — 从填写到云端存储
目标：用 AI 生成带验证的表单，打通"填写 → 提交 → 后台查看"全流程
  • 表单需求描述姓名、手机号、微信号、备注，手机号必填
  • 验证与体验用规则提需求："手机号必须 11 位"；描述成功提示
  • 全流程联调表单提交与云数据库打通，验证端到端数据流通
  • 后台管理在云开发控制台查看用户提交的数据
  • 问题排查检查网络、看报错信息、用 AI 定位原因
  • 动手实操跑通"填写 → 提交 → 后台查看"全流程
第 6 节
测试、发布、上线
目标：完成小程序上线，掌握后续迭代方法
  • 完整测试真机预览，走完所有功能流程，发现问题用 AI 修复
  • 上线流程版本管理 → 提交审核 → 发布 → AppID 配置
  • 持续迭代上线后有新需求？继续用 Vibe Coding 方法描述需求即可
  • 课程总结回顾核心技能，如何将方法迁移到其他项目
  • 动手实操完成提审并上线
W4
答疑周 — 解决真实问题
聚焦：在实际使用中遇到的问题，一对一解决
1v1 答疑
你的问题，现场解决
  • 问题答疑针对上线后遇到的审核问题、用户反馈、功能异常进行 1v1 指导
  • 项目优化根据真实使用数据或反馈，用 Vibe Coding 方法优化现有功能
  • 需求扩展想加新功能？现场演示如何用 Vibe Coding 方法推进
  • 方法复盘总结已内化的技能，规划后续学习方向
